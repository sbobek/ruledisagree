# Structural Rule Disagreement and CC-EDU

This repository provides tools for extracting, comparing, and evaluating local rule-based explanations generated by LUX. The core contribution is the formulation and computation of Structural Disagreement, a similarity-aware measure of conflicting rule behavior, and the Conflict-Conditioned Empirical Disagreement (CC-EDU) score, which quantifies the epistemic instability of rule-based explanations in local neighborhoods of the feature space.

The project includes:
- Computation of rule coverage and structural similarity  
- Local contradiction detection  
- Weighted disagreement scoring  
- Neighborhood-conditioned CC-EDU computation  
- Integration with LUX rule extractors for model explanations  



## Overview

Rule-based explanation methods often generate multiple, potentially conflicting rules for nearby instances. This project provides a principled way to measure such conflicts.



## 1. Structural Disagreement

Structural Disagreement measures how strongly two rules conflict. It combines three factors:

1. Coverage overlap – how much the rules apply to the same instances within a local neighborhood.
2. Feature overlap (Fo) – how similar the rules are in terms of the features they use.
3. Confidence penalty – how strongly and confidently the rules support their respective predictions.

The disagreement score increases when:
- The rules apply to overlapping regions,
- They rely on similar features,
- They make confident but competing predictions.

Default parameter values:
- alpha = 0.3  
- beta = 0.2  

These can be adjusted depending on the application.



## 2. CC-EDU: Conflict-Conditioned Empirical Disagreement

CC-EDU measures how often the rule explaining a given instance conflicts with the rules of its neighboring instances.

For each instance:
- We examine its k-nearest neighbors.
- We detect rule contradictions.
- We weight those contradictions using Structural Disagreement.
- We average the result over the neighborhood.

A high CC-EDU score indicates:
- Dense local conflicts  
- Competing explanations  
- Lower robustness and stability of the explanation mechanism
